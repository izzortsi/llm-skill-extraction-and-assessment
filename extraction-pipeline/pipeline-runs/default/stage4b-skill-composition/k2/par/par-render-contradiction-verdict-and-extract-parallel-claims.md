# Par Render Contradiction Verdict And Extract Parallel Claims

## When to Use

- Multiple skills are needed simultaneously: Render Contradiction Verdict, Extract Parallel Claims
- When a factual comparison between a source claim and an external claim has been completed and a final judgment must be stated.
- When a student's or external claim needs to be checked against a source passage and the relevant statements must first be isolated before any comparison can occur.

## Procedure

1. Apply the following skills in parallel:
2.   - Render Contradiction Verdict: Use when When a factual comparison between a source claim and an external claim has been completed and a final judgment must be stated.
3.   - Extract Parallel Claims: Use when When a student's or external claim needs to be checked against a source passage and the relevant statements must first be isolated before any comparison can occur.
4. Integrate the outputs from all parallel skills:
5.   - Identify complementary results across skills
6.   - Resolve any conflicts between skill outputs
7.   - Synthesize a combined result

## Constraints

- All parallel skills must receive the same input context
- The verdict must be grounded solely in the source passage, not external knowledge.
- Both claims must address the same topic or entity for extraction to be meaningful.
