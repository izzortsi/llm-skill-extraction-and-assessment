# Par Identify Factual Contradiction And Render Contradiction Verdict And Extract Parallel Claims

## When to Use

- Multiple skills are needed simultaneously: Identify Factual Contradiction, Render Contradiction Verdict, Extract Parallel Claims
- When two parallel claims on the same topic have been extracted and the analyst needs to determine whether they agree or conflict.
- When a factual comparison between a source claim and an external claim has been completed and a final judgment must be stated.
- When a student's or external claim needs to be checked against a source passage and the relevant statements must first be isolated before any comparison can occur.

## Procedure

1. Apply the following skills in parallel:
2.   - Identify Factual Contradiction: Use when When two parallel claims on the same topic have been extracted and the analyst needs to determine whether they agree or conflict.
3.   - Render Contradiction Verdict: Use when When a factual comparison between a source claim and an external claim has been completed and a final judgment must be stated.
4.   - Extract Parallel Claims: Use when When a student's or external claim needs to be checked against a source passage and the relevant statements must first be isolated before any comparison can occur.
5. Integrate the outputs from all parallel skills:
6.   - Identify complementary results across skills
7.   - Resolve any conflicts between skill outputs
8.   - Synthesize a combined result

## Constraints

- All parallel skills must receive the same input context
- The comparison must be between claims on the same attribute or relationship.
- The verdict must be grounded solely in the source passage, not external knowledge.
- Both claims must address the same topic or entity for extraction to be meaningful.
